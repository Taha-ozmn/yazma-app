# Yazma

J.A.R.V.I.S. tarzı minimal yazma uygulaması. Metinler otomatik kaydedilir.

## Özellikler

- Otomatik kaydetme (600ms debounce)
- Kelime ve karakter sayacı
- Karanlık tema
- iOS ve Android desteği (Expo)

## Kurulum

```bash
npm install
npm start
```

## Teknolojiler

- React Native
- Expo 52
- AsyncStorage
